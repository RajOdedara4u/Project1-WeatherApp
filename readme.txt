-----------------------------------------------------------  ** Weather Site README ** -----------------------------------------------------------
 
== > Overview:
		- This project is a weather website that displays current weather information based on user input. It fetches data from a weather API to show temperature, wind speed, and other relevant weather details for a specified area.

== > Features:
		- Displays current temperature in Celsius and Fahrenheit.
Shows wind speed in kilometers per hour (km/h).
Provides weather information for any specified location.
Responsive design using HTML, CSS, and JavaScript for a seamless experience on desktop and mobile devices.

== > Usage:
 1) Enter the name of a city or a specific location in the input field.
 2) Click the "Search" button or press Enter to fetch the weather data.
 3) The current temperature in Celsius and Fahrenheit will be displayed, along with wind speed in km/h.

==> Technologies Used:
	- HTML5
	- CSS3 (Flexbox for layout)
	- JavaScript (ES6)
	- Weather API from weather.visualcrossing.com
